Introdução

A construção de um website básico é um passo fundamental no mundo da criação web. Para alcançar esse objetivo, é essencial compreender as tags HTML e como utilizá-las de forma eficaz. Nesta matéria, exploraremos os conceitos fundamentais por trás das tags HTML e como elas são utilizadas para estruturar e formatar o conteúdo de uma página web.

Fundamentos do HTML

voltar

HTML, ou Hypertext Markup Language, é a linguagem padrão para criação e formatação de páginas web. Desenvolvida por Tim Berners-Lee, o inventor da World Wide Web, a linguagem HTML permite que os desenvolvedores criem estruturas claras e acessíveis para o conteúdo online. Nesta seção, examinaremos os conceitos básicos do HTML e sua importância na construção de websites.

Tags Básicas do HTML

voltar

As tags HTML são elementos fundamentais que definem a estrutura e o estilo de uma página web. Desde cabeçalhos e parágrafos até listas e links, cada tag desempenha um papel específico na organização do conteúdo. Nesta parte, detalharemos as funções de cada uma das tags mencionadas no índice, acompanhadas de exemplos práticos de seu uso

  • < h1 > até < h6 >

      São tags de cabeçalho que definem a importância e hierarquia do texto. O h1 é o cabeçalho de maior importância, enquanto o h6 é o de menor importância.

      Cabeçalho 1

      Cabeçalho 2

      Cabeçalho 3

      Cabeçalho 4

      Cabeçalho 5
      Cabeçalho 6

  • < p >

      A tag < p > é usada para definir parágrafos em HTML, separando o conteúdo textual em blocos distintos.

  • < mark >

      A tag < mark > é usada para marcar o texto em um lugar específico. ex Texto marcado

  • < small >

      A tag < small > é usada para definir o texto em uma fonte pequena. ex Texto pequeno

  • < i >

      A tag < i > é usada para definir o texto em uma fonte italica. ex Texto italico

  • < u >

      A tag < u > é usada para definir o texto em uma fonte sublinhada. ex Texto sublinhado

  • < strong >

      A tag < strong > é usada para definir o texto em uma fonte em negrito. ex Texto em negrito

  • < ol >

      A tag < ol > é usada para definir uma lista numerada. ex

      1. Item 1
      2. Item 2
      3. Item 3
  • < ul >

      A tag < ul > é usada para definir uma lista não numerada. ex

          Item 1
          Item 2
          Item 3
    • < li >

        A tag < li > é usada para definir um elemento da lista. ex

      • Item 1
      • Item 2
      • Item 3
    • < a >

        A tag < a > é usada para definir um link. ex Link Google

    • < hr >

    • < sub >

        A tag < sub > é usada para definir um texto subscrito. ex Texto subscrito

    • < sup >

        A tag < sup > é usada para definir um texto superscripto. ex Texto superscripto

    • < blockquote >

        A tag < blockquote > é usada para definir um bloco citado. ex

        Texto citado
    • < font >

        A tag < font > é usada para definir uma fonte. ex Texto em vermelho

    • < del >

        Utilizada para marcar texto que foi deletado ou removido de uma versão anterior do documento.
